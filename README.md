# Resumo de Livros

Site pessoal para publicar resumos dos livros que estou lendo.

## Sobre

Um espaço para registrar aprendizados e compartilhar leituras, construído com HTML/CSS/JS puro e hospedado no GitHub Pages.

## Documentação

Veja as especificações completas do projeto em [`docs/especificacoes.md`](docs/especificacoes.md).

## Como adicionar um livro

1. Criar a pasta `livros/<slug-do-livro>/`
2. Adicionar `capa.jpg` e `resumo.md`
3. Adicionar o livro em `data/livros.json`
4. Commit e push
